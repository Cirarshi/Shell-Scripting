## 🚀 Getting Started

### Prerequisites

- Linux / Unix system (or WSL / Git Bash)
- Basic terminal knowledge
- Bash installed

---

### 🔧 How to Run

1. Clone the repository:
```bash
git clone https://github.com/Cirarshi/Shell-Scripting.git
cd Shell-Scripting
```

2. Give execute permission:
```bash
chmod +x script_name.sh
```

3. Run the script:
```bash
./script_name.sh
```
